# Disconnect a SemsqlConn from its database

Disconnect a SemsqlConn from its database

## Usage

``` r
disconnect(x, ...)
```

## Arguments

- x:

  A `SemsqlConn` object.

- quiet:

  logical(1) if TRUE suppresses the disconnection message (default
  FALSE).

## Value

The `SemsqlConn` object invisibly.
