# a named vector with mapping from cell type phrase to CURIE for CL.owl of 2025-12-17

a named vector with mapping from cell type phrase to CURIE for CL.owl of
2025-12-17

## Usage

``` r
data(cn2tag)
```

## Format

names character vector

## Examples

``` r
data("cn2tag", package = "ontoProc2")
cn2tag["Kupffer cell"]
#> Kupffer cell 
#> "CL:0000091" 
```
