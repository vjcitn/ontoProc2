# ontoProc2: Ontology Facilities Based on INCAtools Semantic SQL

This package provides ontology facilities based on INCAtools Semantic
SQL. Tooling is provided to retrieve and cache SQLite databases
representing curated ontologies. ontologyIndex ontology_index instances
can be produced. Anthropic's Claude was used in the development of S7
classes and methods and in the production of documentation.
